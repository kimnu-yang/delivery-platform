## Intro
: &nbsp; 해당 프로젝트는 자바 스프링부트를 이용한 배달 플랫폼의 API 서버를<br>
프로젝트 하위에 User Api, Store Api, Common, DB 각각의 모듈로 나누어 개발한 멀티 모듈 구조의 스프링부트 프로젝트입니다.

## Project Summary
- 개발언어
  - Java 17
- 프레임워크
  - Spring Boot 3.2.0
- 데이터베이스
  - MySQL 8.0.2
- ORM
  - JPA
- 메시지 큐
  - RabbitMQ 3.1.2
- 웹 인증 방식
  - Header: Json Web Token
- API 명세
  - Swagger(Springdoc)
